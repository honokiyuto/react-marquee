# React Marquee

HTMLの`<marquee>`要素を模したモダンなReactコンポーネントです。テキストや要素を様々な方向にスクロール表示させることができます。

## 特徴

- 🎯 **TypeScriptで型安全** - 完全な型定義で開発体験が向上
- 🎨 **複数のbehaviorモード** - scroll, slide, alternateの3つのモード
- 🧭 **4方向のスクロール対応** - left, right, up, downに対応
- ⚡️ **カスタマイズ可能** - スピードとループ回数を自由に調整
- 🎭 **Storybook完全対応** - 豊富なサンプルとドキュメント

## クイックスタート

```tsx
import { Marquee } from './Marquee';

function App() {
  return (
    <Marquee behavior="scroll" direction="left">
      スクロールするテキスト
    </Marquee>
  );
}
```

## 基本的な使い方

### Scrollモード（デフォルト）

```tsx
<Marquee direction="left">左にスクロールします</Marquee>
```

### Slideモード

```tsx
<Marquee behavior="slide" direction="right">
  右にスライドします
</Marquee>
```

### Alternateモード

```tsx
<Marquee behavior="alternate" direction="left">
  左右に往復します
</Marquee>
```

## スピードの調整

スクロールスピードは`scrollAmount`（移動ピクセル数）と`scrollDelay`（インターバル）で調整できます：

```tsx
{
  /* 高速 */
}
<Marquee scrollAmount={20} scrollDelay={60}>
  高速スクロール
</Marquee>;

{
  /* 低速 */
}
<Marquee scrollAmount={2} scrollDelay={200}>
  低速スクロール
</Marquee>;
```

## ループ回数の制御

デフォルトは無限ループ（`-1`）ですが、回数を指定することもできます：

```tsx
{
  /* 3回だけスクロール */
}
<Marquee loop={3}>3回だけスクロールして停止</Marquee>;
```

## スタイリング

`style`プロパティや`className`でカスタマイズできます：

```tsx
<Marquee
  style={{
    backgroundColor: '#4CAF50',
    color: 'white',
    padding: '20px',
    fontSize: '24px',
  }}
>
  カスタムスタイル
</Marquee>
```
